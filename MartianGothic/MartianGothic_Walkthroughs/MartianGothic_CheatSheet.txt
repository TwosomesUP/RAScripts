Martian Gothic - One Page Cheat Sheet

== Codes & Passwords ==
- PANDORA → Director’s computer
- PARK LANE → Ben Gunn note / MOOD password
- WORMWOOD → Bible (Gym)
- Watch codes: Karne 4864 | Kenzo 3172 | Matlock 2915
- Erebus printout → Necropolis elevator

== Tags & Doors ==
- Orange Tag → Felicci → Director’s Quarters
- Green Tag → Park Lane hatch (4864) → Gym, Washroom, Rec Room
- Blue Tag → Study desk, Arboretum zombie, lockers → Blue Doors
- Brown Tag → Piccadilly Circus → Chem Lab
- Yellow Tag → Piccadilly east corpse → MOOD chamber
- Red Tag → Trade Ben Gunn → Bio Lab
- White Tag → Bio Lab front → Mai Lin’s Quarters
- Black Tag → Mai Lin’s → Necropolis elevator
- Rainbow Tag → Harroway’s quarters → Kremlin
